Scientific Symbols (Chrome)

This extensions allows you to easily copy and paste custom symbols into your writing. By default there are some math symbols, but you can add anything you want through the "custom" section.

You can download the extension through the chrome store by clicking the link above, or alternatively you can download the unpacked extension to view the source code and add it by enabling "developer mode" on chrome://extensions by clicking here.

NOTE! This extension is for Google Chrome.
