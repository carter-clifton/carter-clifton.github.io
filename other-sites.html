Character Counter

This website counts how many characters you have typed and if you go over a certain threshold (Set to 1000 by default) the screen will turn red indicating you have gone over the limit. This can be useful when you are required to stay within a certain number of characters for a particular writing piece.

The code of this website can be found on github.

BF Interpreter

This website allows you to run BF code in the browser. Additionally, you can view the memory addresses to see how your code influences the values in memory. The entries at the top of the page allow you to change the maximum value in each memory address and the number of memory addresses.

The code of this website can be found on github.

NOTE! Character input using "," is not currently supported.
